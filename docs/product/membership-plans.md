## 2. 开通方式

- `enableProductionPayments` 由生产环境变量控制；开启后 Plus/Pro 通过 Plisio 创建支付 invoice。
- 结算金额按 USD 会员价格生成；用户可用 Tether BEP-20 (`USDT_BSC`) 或 Tether ERC-20 (`USDT`) 支付。
- Plisio 商户设置为平台承担佣金，不向用户额外转嫁支付佣金。
- 支付完成以 Plisio 服务端回调为准；回调必须验签成功后才开通会员。
- 管理员人工开通、内部邀请码或测试 fixture 仍可用于运营和客服处理；所有管理端变更写审计日志。
- 当前不提供自动续费；退款和税务开票流程需按运营规则人工处理并留痕。

## 3. 权益规则

1. 权益按订阅有效期计算；过期后回到 Free。
2. 降级导致超配额时，不删除历史对象；禁止新增，并要求用户暂停/结束到配额内。
3. 已创建模拟盘仍受策略下架、行情异常和风控暂停约束。
4. 管理员赠送或延长必须填写原因、起止时间和工单/活动来源。
5. 会员权益只销售访问、数据、提醒和模拟容量，不承诺任何收益。
